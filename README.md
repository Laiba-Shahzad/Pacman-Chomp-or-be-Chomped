# Pacman-Chomp-or-be-Chomped
A classic Pacman clone built from scratch using C++ and SFML.

## Features
- Full playable maze
- Ghost AI
- Scoring system
- Sound effects

## Built With
- C++
- SFML
- Visual Studio

## How to Build
1. Clone this repo
2. Open project in Visual Studio
3. Set to Release mode
4. Make sure SFML is linked properly
